# draw-heart-wgpu

Render a heart in a native window, using [winit](https://github.com/rust-windowing/winit) and [wgpu](https://github.com/gfx-rs/wgpu).

## Getting Started

- update `VERTICES`:
  - run `python3.11 ./scripts/gen_vertices.py >./src/assets/vertices.rs`
- run:
  - run `cargo run`
